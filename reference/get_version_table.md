# Get released versions of ASReml-R in lookup table

Get released versions of ASReml-R in lookup table

## Usage

``` r
get_version_table(
  url =
    "https://asreml.kb.vsni.co.uk/asreml-r-4-download-success/?site_reference=VS9AF20"
)
```

## Value

A list of data frames containing the version number and release date of
released ASReml-R versions for comparison

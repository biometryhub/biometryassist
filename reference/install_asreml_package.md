# Install the ASReml package

Install the ASReml package

## Usage

``` r
install_asreml_package(save_file, library, quiet, os, verbose = FALSE)
```

## Arguments

- save_file:

  Path to package file

- library:

  Library path

- quiet:

  Whether to suppress messages

- os:

  Operating system

- verbose:

  Logical for verbose output

## Value

TRUE if successful, FALSE otherwise
